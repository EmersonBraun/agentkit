# 示例

展示 AgentsKit 能构建什么的交互式演示。每个示例都是聊天——这正是 AgentsKit 所擅长的。

## 聊天模式

| 示例 | 展示内容 |
|---------|---------------|
| [基础聊天](./basic-chat) | 流式对话、停止按钮、键盘处理 |
| [工具使用](./tool-use) | 函数调用——天气、搜索，可展开结果卡片 |
| [多模型](./multi-model) | Claude 与 GPT 并排对比 |
| [代码助手](./code-assistant) | 带语法高亮的流式代码 |
| [客户支持](./support-bot) | 快捷回复、输入中指示、升级流程 |
| [RAG 聊天](./rag-chat) | 文档上下文、引用、来源 |
| [智能体操作](./agent-actions) | AI 生成交互式实时 UI（任务看板、仪表盘） |
| [Markdown 聊天](./markdown-chat) | 富文本——表格、列表、代码块、引用 |

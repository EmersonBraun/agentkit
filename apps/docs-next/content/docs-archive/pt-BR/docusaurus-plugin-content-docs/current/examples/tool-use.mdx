# Uso de ferramentas

Assistentes de IA que chamam funções — clima, busca na web, consultas a banco. O AgentsKit renderiza chamadas de ferramenta com cards expansíveis mostrando argumentos e resultados.

<ToolUseChat />

## Com o AgentsKit

O componente `ToolCallView` cuida de tudo:

```tsx
import { useChat, ChatContainer, Message, ToolCallView } from '@agentskit/react'

function Chat() {
  const chat = useChat({ adapter })
  return (
    <ChatContainer>
      {chat.messages.map(msg => (
        <div key={msg.id}>
          <Message message={msg} />
          {msg.toolCalls?.map(tc => (
            <ToolCallView key={tc.id} toolCall={tc} />
          ))}
        </div>
      ))}
    </ChatContainer>
  )
}
```

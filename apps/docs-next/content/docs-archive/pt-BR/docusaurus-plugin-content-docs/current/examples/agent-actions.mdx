# Ações do agente

Agentes de IA que geram UI interativa ao vivo — rastreadores de tarefas, dashboards, formulários. O agente não só responde com texto, ele constrói interfaces funcionais.

<AgentActions />

## Com o AgentsKit

Combine `useChat` com renderizadores customizados para resultados de chamadas de ferramenta:

```tsx
import { useChat, ChatContainer, Message } from '@agentskit/react'

function AgentChat() {
  const chat = useChat({ adapter })
  return (
    <ChatContainer>
      {chat.messages.map(msg => (
        <div key={msg.id}>
          <Message message={msg} />
          {msg.toolCalls?.map(tc => (
            <DynamicWidget key={tc.id} type={tc.name} data={JSON.parse(tc.result || '{}')} />
          ))}
        </div>
      ))}
      <InputBar chat={chat} />
    </ChatContainer>
  )
}
```

# Comece aqui (~60 segundos)

1. **[Início rápido](./quick-start)** — copie e cole um chat com streaming; troque o adaptador em uma linha.
2. **[Visão geral dos pacotes](../packages/overview)** — os quatorze pacotes `@agentskit/*` e links para cada guia.
3. **Abra o guia do que você entrega** — React, Ink, runtime, RAG etc., a partir dessa tabela.

**Assinaturas e tipos:** [API gerada (TypeDoc)](pathname:///agentskit/api-reference/) (execute `pnpm --filter @agentskit/docs docs:api` uma vez se usar `docusaurus start` localmente).

**Cola de instalação:** [Instalação](./installation).

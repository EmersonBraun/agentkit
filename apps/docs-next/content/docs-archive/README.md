# Archived translations

These trees (`es/`, `pt-BR/`, `zh-Hans/`) are the Docusaurus-era translations from the retired `apps/docs` site, preserved verbatim so nothing is lost during the Fumadocs migration.

They are **not** wired into the Fumadocs sidebar. Do not expect them to reflect current English docs — see [RFC 0001 (i18n strategy)](../../../../rfcs/0001-i18n-strategy.md) for the plan.

If you want to restore a locale, pick it up from here and port into Fumadocs under the chosen i18n strategy.

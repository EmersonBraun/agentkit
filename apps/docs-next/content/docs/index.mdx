AgentsKit is a family of small, plug-and-play packages that cover the entire
agent lifecycle in JavaScript: chat UIs, autonomous runtimes, tools, skills,
memory, RAG, observability.

This documentation site is a Fumadocs spike — it sits in parallel to the
existing Docusaurus site at `apps/docs` while we evaluate the migration.

## The substrate

Six core contracts (formalized as ADRs) define how every package composes:

| Contract | Role |
| --- | --- |
| Adapter | LLM provider seam |
| Tool | Function the model calls |
| Memory | Chat history + vector store |
| Retriever | Context fetching |
| Skill | Declarative persona |
| Runtime | The loop that composes them all |

If you understand these six, you understand AgentsKit.
